Короче, по api:



/registration 

принимает логин и пароль, возвращает что-то типа ок



/login 

принимает логин и пароль, возвращает jwt токен, в дальнейшем я пихаю его через header (из него можно понять кто сделал запрос)



/create_place

 принимает координаты и название места, возвращает что-то типа ок и записывает место в базу



/create_parking 

принимает координаты и количество мест на парковке, возвращает что-то типа ок и записывает парковку в базу



/get_places

 возвращает список мест пользователя



/get_nearest_parkings 

принимает id места пользователя и промежуток времени и возвращает список ближайших парковок с местами (пусть не больше 5)



/book_place принимает 

id парковки и промежуток времени и создает в базе бронирование места на парковке для пользователя



/get_books 

возвращает - список бронирований пользователя
